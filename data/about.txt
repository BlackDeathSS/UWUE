Bunker Blaster by Lance Refior

Controls:
WASD - walk
Shift - run
Space - jump
Arrow keys - look around
E - interact
ESC - back
Enter - confirm
F2 - save screenshot
F11 - toggle fullscreen

This game uses these Java libs:
Lightweight Java Game Library
Java OpenGL Math Library
Luaj
